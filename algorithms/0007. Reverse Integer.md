# 数学反转
* 时间复杂度：O(log<sub>10</sub>(N))
* 空间复杂度：O(1)
```java
public int reverse(int x) {
    int rev = 0;
    while(x != 0) {
        int num = x % 10;
        x /= 10;
        // x 是正数，那么反转后个位数最大为7
        if (rev > Integer.MAX_VALUE / 10 || rev == Integer.MAX_VALUE && rev > 7) {
            return 0;
        }
        // x 是负数，那么反转后个位数最大为8
        if (rev < Integer.MIN_VALUE / 10 || rev == Integer.MIN_VALUE && rev < -8){
            return 0;
        }
        rev = rev * 10 + num;
    }
    return rev;
}
```
